# ☕ CoffeeCartel — Premium Coffee Brand Website

An interactive and visually immersive coffee brand website built using modern frontend technologies. The project focuses on delivering a premium digital experience through animations, responsive layouts, and interactive UI components inspired by modern startup landing pages.

## 🚀 Project Overview

CoffeeCartel is a modern frontend web project designed to simulate a professional coffee brand’s online presence. The website emphasizes user engagement through smooth animations, 3D visual effects, and clean UI/UX principles.

The goal of this project was to build a production-style landing page showcasing advanced frontend development skills and modern design trends.

## 🛠 Tools & Technologies

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Responsive Web Design
- CSS Animations & Transitions
- Intersection Observer API

## 📂 Project Structure

The project includes:

- `index.html` — Website structure and layout
- `styles.css` — Styling, animations, and responsive design
- `scripts.js` — Interactive behavior and dynamic effects
- Modern UI components and animated sections

## 📊 Key Features

### 🔹 Interactive Hero Section
- Animated coffee visuals
- Mouse-based motion effects
- Modern gradient and glassmorphism styling

### 🔹 Smooth Scroll Animations
- Elements animate when entering viewport
- Implemented using Intersection Observer API

### 🔹 Responsive Design
- Optimized for desktop and mobile devices
- Flexible layouts using modern CSS techniques

### 🔹 Dynamic UI Effects
- Hover animations and transitions
- Floating and motion-based elements
- Real-time footer year update using JavaScript
